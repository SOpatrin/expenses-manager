# Roadmap

Фичи режем вертикально: lib → actions → UI, коммит после каждого слоя.
Авторизация — последней, когда нужно деплоить.

## М1 — Основной поток

- [ ] **Add transaction** — форма (расход/доход), `useActionState`, `useOptimistic`
- [ ] **Transaction list** — RSC-список за текущий месяц
- [ ] **Stats / Dashboard** — баланс, сумма трат, топ-категории

## М2 — Полезность

- [ ] **Фильтры** — по категории, типу, дате
- [ ] **Редактирование / удаление** транзакции

## М3 — Многопользовательность

- [ ] **Auth** — Google OAuth, защита роутов
- [ ] **Wallet init** — автосоздание при первом входе
- [ ] **Шаринг кошелька** — пригласить по email

## М4 — Расширение

- [ ] **Telegram bot** — вебхук → вызов lib/ → ответ
